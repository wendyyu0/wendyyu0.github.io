Projects (Computer Science)

Blockchain for Medical Devices

The purpose of this project was to create a webapp backed with blockchain technology for patients and doctors to interact with each other securely. I completed the webapp portion using Node.js/Bootstrap for front end, Express for middleware, Passport for password security, and MongoDB for backend. The Ethereum blockchain platform was used, in addition to GoLang for interacting with the webapp.

Hypatia Catalog

The Hypatia Catalog is a one of a kind, largest compilation of stellar abundancies. It is named after Hypatia, one of the first documented female astronomers born between 350-375AD. I imported the data from 20,000 stars into a SQLite database using Python, and this data is displayed on the (in-progress) website using Web2Py.

Sentiment Analysis Project

This project was done for a hackathon. The purpose is to do a Google search of a specific stock, and runs the found pages through the program to determine if a stock is bearish or bullish by utilizing sentiment analysis. We implemented the part of speech tagger using NTLK for Python. We attempted to modify the PMI-IR algorithm to estimate the semantic orientation of the extracted phrases. (Disclaimer: the algorithm didn't work that well, but it was a good learning experience.) This sparked my interest in machine learning, and I hope to work more on this topic in the future.

Deployment and Orchestration Infrastructure

This project was a study on using sockets and writing in low-level code. The shell interface is provided by the master deployment manager, which delegates the commands and processes to the worker deployment managers. I used C++ and used the TCP Socket to accomplish the orchestration.

Projects (Chemical Engineering)

Gold Nanoparticle Research

The research's focus was to improve drug delivery (specifically chemotherapy) in nanomedicine using gold nanoparticles. The idea was that the gold nanoparticles provided a shell around the medicine due to polarities and hydrophobicity, and thus the medicine could be safely transported to the cancerous site without affecting the other healthy organs.

Tissue Engineering Research

I synthesized a hybrid hydrogel composed of methacrylated gelatin and methacrylated tropoelastin. I did data analyses on the hydrogels, and I used a mechanical tester, an SEM microscope, FTIR/H NMR analysis, and live/dead assay to characterize and analysize these hydrogels. The idea is to create an injectable hyaluronic acid/elastin-like polypeptide hybrid for easy, noninvasive cartilage repair.
